Tandem Internal Tools

Infrastructure Self-Service Portal

Streamlined infrastructure management for Tandem Diabetes Care

Login to Access

To use this application, you need a GitHub Personal Access Token with the following scopes:

  • repo - Full control of private repositories
  • workflow - Permission to trigger GitHub Actions workflows
  • read:org - To verify organization and team membership

How to generate a token:

  1. Go to GitHub Personal Access Tokens
  2. Enter a note like "Infrastructure Self-Service Tool"
  3. Select the required scopes listed above
  4. Click "Generate token"
  5. Copy the generated token and paste it below

Token is securely transmitted and not stored permanently.
